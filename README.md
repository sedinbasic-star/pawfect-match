Pawfect Match – Final Project

This repository contains the full source code for the Pawfect Match website, created as part of the final class project.

How to Run the Website
1. Download the repository by clicking the green Code button and selecting Download ZIP.
2. Extract the ZIP file to your computer.
3. Open the project folder.
4. Double-click the file landing_Page.html to launch the website in your browser.
No additional setup or installation is required. The website will run locally on any modern browser.

Project Structure
landing_Page.html – Home page
adoption_Page.html – Adoptable pets
how_to_adopt.html – Adoption steps
resources.html – External resources
about_contact.html – About & contact form
book_adoption_visit.html – Adoption appointment form
images/ – Contains all site images
styles.css – Website styling
script.js – JavaScript for form handling and interactions

Notes
The contact and appointment forms use Web3Forms to send notifications.
Best viewed on desktop.
